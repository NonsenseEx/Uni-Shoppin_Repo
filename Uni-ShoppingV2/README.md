# Uni-Shoppin 🎮
> **Dominio total en cada click.**

## 📝 Descripción del Proyecto
**Uni-Shoppin** es una aplicación web funcional diseñada para la gestión de inventario y exhibición de artículos electrónicos de alto rendimiento, periféricos y hardware enfocado en el ecosistema "gamer". El objetivo principal es ofrecer una plataforma dinámica donde el usuario pueda interactuar con un catálogo de productos de forma segura y eficiente.

Este proyecto ha sido desarrollado aplicando conceptos avanzados de **JavaScript**, con un fuerte enfoque en la manipulación dinámica del DOM, el uso de estructuras de datos (arreglos y objetos) y la implementación de funciones modulares.

## 🚀 Características Principales
* **Gestión de Catálogo**: Visualización dinámica de los artículos almacenados en memoria, permitiendo la actualización constante de la interfaz.
* **Formulario de Registro**: Interfaz para añadir nuevos periféricos al catálogo con validaciones integradas.
* **Carrito de Compras**: Sistema interactivo para la selección de productos y cálculo de totales.
* **Interfaz Responsiva**: Diseño moderno con estética "gamer", incluyendo efectos visuales como gradientes y desplazamiento parallax.

## 🛠️ Tecnologías Utilizadas
* **HTML5**: Estructura semántica para la aplicación.
* **CSS3**: Estilos personalizados, variables de color y layouts dinámicos.
* **JavaScript **: Lógica de negocio, gestión de eventos y manipulación del DOM.

## 🔒 Seguridad y Buenas Prácticas
En cumplimiento con los requisitos de seguridad para aplicaciones web, se implementaron las siguientes medidas:
* **Prevención de XSS**: Se evita el uso de `innerHTML` para la inserción de datos sensibles, priorizando métodos como `textContent` y `createElement`.
* **Sanitización de Datos**: Validación y limpieza de las entradas de los formularios.
* **Código Modular**: Uso de funciones reutilizables que favorecen la claridad y el mantenimiento del código.

## 🤖 Uso de Inteligencia Artificial
Para potenciar el desarrollo de **Uni-Shoppin**, se integraron herramientas de IA siguiendo los lineamientos académicos:
* **Refactorización**: Optimización de la lógica de renderizado del catálogo para mejorar el rendimiento.
* **Validaciones**: Generación de expresiones regulares para la validación robusta de campos técnicos.
* **Documentación**: Apoyo en la estructuración técnica y redacción de este archivo.

## 📦 Instalación
Para ejecutar el proyecto de forma local:
1.  Clona el repositorio en tu máquina local.
2.  Navega a la carpeta del proyecto y abre el archivo `index.html` en tu navegador preferido.
